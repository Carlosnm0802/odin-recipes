# Odin Recipes

Proyecto introductorio de The Odin Project en el que construyo un sitio web simple de recetas usando HTML. El objetivo es practicar la estructura básica de una página web y la organización de archivos.
Para este proyecto me propuse en crear un recetario web básico para practicar la construcción de páginas estáticas con HTML.

## Habilidades demostradas
- Estructura HTML semántica (headings, párrafos, listas, enlaces e imágenes).
- Navegación entre páginas con enlaces relativos.
- Organización del proyecto en carpetas y archivos.
- Uso básico de Git y GitHub para control de versiones.

## Tecnologías
- HTML5
- Git
- GitHub

## Reflexión final
Al finalizar este proyecto reforcé mis bases de HTML y entendí mejor cómo organizar un sitio estático desde cero. También practiqué el flujo de trabajo con Git, haciendo commits más claros y frecuentes. Como siguiente paso, me gustaría añadir estilos con CSS para mejorar la presentación y hacer el sitio más atractivo.
